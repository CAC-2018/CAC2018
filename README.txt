<08-04-2016|ND>: To update this file, put your updates after a tag in 
	the format "<MM-DD-YYYY|FirstInitialLastInitial>:". Indent with
	TAB to continue your text. bin is the output folder, src is the
	source (primary build) folder, and tst is the test folder.
<08-05-2016|ND>: res is the resource folder used for additional 
	non-code files that will be part of the app. When building the
	app, make changes to the test class and if they are satisfactory,
	update them to the main class.
